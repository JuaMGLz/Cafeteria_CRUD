# ☕ Cafetería CRUD

Sistema de gestión web para una cafetería, enfocado en la administración de productos y pedidos mediante operaciones CRUD (Create, Read, Update, Delete). 

Este proyecto demuestra los fundamentos del desarrollo web dinámico y la gestión de datos, habilidades esenciales para la construcción de interfaces interactivas y escalables.

## 🚀 Características Principales

- **Gestión de Inventario (CRUD):** 
  - **Crear (Create):** Registro de nuevos productos (ej. bebidas, postres) con sus respectivos precios y descripciones.
  - **Leer (Read):** Visualización del menú y catálogo de productos disponibles en tiempo real.
  - **Actualizar (Update):** Modificación de precios, stock o detalles de productos existentes.
  - **Eliminar (Delete):** Remoción de artículos del sistema de manera segura.
- **Interfaz Dinámica:** Interacción fluida con el usuario mediante actualizaciones en pantalla sin necesidad de recargar la página.
- **Lógica de Datos:** Manejo de la información y estados de la aplicación a través de estructuras de datos en memoria o almacenamiento local.

## 🛠️ Tecnologías y Herramientas

- **Lenguaje Principal:** JavaScript (Lógica de negocio y manipulación del DOM).
- **Frontend:** HTML5 y CSS3 (Estructuración y diseño de la interfaz).
